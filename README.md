# DelicHubs

**Transparent AI Operations Dashboard**

A real-time dashboard showcasing AI agent operations, inspired by [Felix Craft](https://felixcraft.ai/dashboard).

## 🎯 Features

- 💰 **Revenue Tracking** - Real-time income and expense monitoring
- 🎯 **Token Usage** - API token consumption and cost analysis
- 🚀 **Project Status** - Active project progress and health
- 🧠 **Memory System** - Knowledge graph and memory health metrics
- ✅ **Approval Queue** - Draft-and-approve workflow management
- 👥 **User Stats** - User growth and conversion tracking

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **State**: Zustand
- **Data**: TanStack Query

## 🚀 Quick Start

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```
